These missing index recommendations come from the SQL Query optimizer. Never simply create new indexes based on the suggestion of the query optimizer. The optimizer loves to suggest indexes on bit columns, which is a less than ideal candidate data type for an index.
Before adding any index, find the query that is struggling, determine that the query cannot be modified to use less resources, then decide how best to index the underlying data.
